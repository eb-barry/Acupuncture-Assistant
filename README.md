# 針灸助理 Acupuncture Assistant

> 全人醫療・調和陰陽・扶正祛邪

一款以 HTML + JavaScript 為基礎的手機端 PWA 應用程式，涵蓋中醫針灸的三大核心功能模組，專為中醫師、針灸學習者及學術探討設計。

## ✨ 功能概覽

### 1. 靈龜八法與子午流注

依據天干地支演算，提供當前時辰的開穴建議。

**依時取穴**
- 自動依據當前日期與時間，計算天干地支日柱、時柱
- 依靈龜八法公式（陽日 ÷9、陰日 ÷6）演算餘數，對應八卦及主穴、配穴
- 支援自訂日期時間查詢
- 餘數為 5 時提供男／女性別切換（男：照海/列缺；女：內關/公孫）
- 即時秒針時鐘，換時辰自動更新

**依穴取時**（逆查功能）
- 使用者選擇八個主穴之一，系統列出當日所有符合該穴位的開穴時辰
- 每個時辰顯示干支、卦象、穴位角色（主穴／配穴）及配伍穴位
- 標示當前時辰
- 點選穴位即顯示穴位圖與詳細說明

### 2. 經絡與穴位

提供四種查詢模式，靈活適應不同使用習慣：

| 模式 | 說明 |
|------|------|
| 輸入穴位 | 直接輸入穴位名稱，支援即時 Autocomplete |
| 依經脈 | 選擇十四經脈 → 穴位特性 → 穴位名稱，三層連動 |
| 依特性 | 選擇穴位特性（如合穴、八脈交會穴），列出所有符合穴位及所屬經脈 |
| 依病症 | 輸入病症關鍵字，搜尋主治及現代醫學闡釋，列出相關穴位 |

查詢結果顯示：
- 全寬穴位圖（WebP 格式，多 CDN 備援）
- 所屬經脈標籤 + 所有相關穴位特性標籤
- 結構化說明：主治、現代醫學闡釋、取穴要領、簡易取穴法、針灸禁忌

### 3. 中醫歌訣

- 9 首經典中醫歌訣（醫學三字經、四總穴歌、十問歌等）
- 顯示歌訣說明與全文
- TTS 語音朗讀，支援中文女聲／男聲切換
- 可暫停、停止播放

## 🔧 技術架構

| 資料格式 | JSON（穴位資料、歌訣清單），WebP 圖片 |
| 部署平台 | GitHub Pages |

### 計算公式

```
總和 = 日干值 + 日支值 + 時干值 + 時支值
陽日（甲丙戊庚壬）：餘數 = 總和 ÷ 9（整除取 9）
陰日（乙丁己辛癸）：餘數 = 總和 ÷ 6（整除取 6）
```

## 🚀 本機開發

此專案為純靜態網頁，無需安裝任何套件：

```bash
git clone https://github.com/eb-barry/Acupuncture-Assistant.git
cd Acupuncture-Assistant
# 使用任意靜態伺服器，例如 VS Code Live Server 或：
python3 -m http.server 8080
# 開啟 http://localhost:8080
```

> **注意：** Service Worker 需在 HTTPS 或 `localhost` 環境下才能正常運作。
